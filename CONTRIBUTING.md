# Contributing to La Estrella

We welcome contributions to the La Estrella project! To ensure a smooth collaboration, please follow these guidelines:

## Getting Started
1. **Fork the repository**: Click the "Fork" button at the top right of the repo.
2. **Clone your fork**: Clone your copy of the repo to your local machine.
   ```bash
   git clone https://github.com/your-username/laestrella.git
   ```
3. **Create a branch**: Create a new branch for your contribution.
   ```bash
   git checkout -b your-feature-branch
   ```

## Guidelines for Contributions
- **Code Style**: Follow the existing code style and conventions used in the project.
- **Documentation**: Update documentation if your changes require it.
- **Commit Messages**: Make your commit messages descriptive and clear.
- **Pull Requests**: When you are ready, submit a pull request. Provide a clear description of your changes and why they are necessary.

## Reporting Issues
If you encounter any issues, please use the GitHub issue tracker to report them. Be sure to include relevant information to help us understand the problem.

## Thank You!
Thank you for considering contributing to La Estrella! Your support and feedback help us improve the project.
